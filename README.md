# Healthcare Big Data Project

Dự án phân tích và trực quan hóa Health Care Dataset sử dụng Hadoop + Ruby + Chart.js.

## Cấu trúc nhóm
- Member 1: Hadoop Engineer
- Member 2: Ruby Core Developer
- Member 3: Data Analyst
- Member 4: Visualization Expert

## Khởi động
1. Bundle install: `bundle install`
2. Đổi tên `.env.example` thành `.env` và config lại
3. Chờ Member 1 upload data và tạo Hive tables
4. Chạy hệ thống: `ruby orchestrator.rb`
